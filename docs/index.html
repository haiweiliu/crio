Crio - Event Discovery App

About the App

Crio is a React Native mobile application for discovering and managing events. This website serves as a landing page while the web version is under development.

Features

  • Discover events near you
  • Filter by category
  • Search for specific events
  • View featured events

Development

Crio is built with React Native and Expo. The mobile app is currently in active development.
